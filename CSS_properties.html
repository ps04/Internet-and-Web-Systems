CSS Attributes

Background Properties

  1. background-color: color name/hex value/rgb value
  2. background-image: url("path to image")
  3. background-repeat: no-repeat/ repeat - y/repeat - x/repeat

Spacing Scheme

  1. width: specify width in pixels or percentage
  2. breadth: specify breadth in pixels or percentage
  3. border: specify style(dotted, solid etc)| width| color
  4. padding: specify the gap between the content and the html element border
  5. margin: specify the gap between different html elements, as margin-top/-bottom/-right/-left

Font

  1. color: color name/hex value/rgb value
  2. font-size: specify size in pixels / percentage
  3. font-family: specify multiple font names to select the first one available from the list
  4. font-style: italic/ normal/ oblique

Text

  1. text-align: specify alignment of text in the lement, like right/left/center/inherit from parent / justify to container or section width .
  2. text-transform: specify case sensitivity of text, like uppercase/ lowercase
  3. letter-spacing: specify spacing in pixels / percentage (parent retalive).
  4. line-height: specify line spacing in pixels / percentage .

Float

  1. float: left/right
  2. float is different from align as align will justify the content in the element, whereas float justifies the element itself
